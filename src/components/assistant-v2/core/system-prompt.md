## 响应规则

JSON 规范： <tool> 标签内的内容必须是合法的 JSON 格式。

JSON 结构： 必须严格遵守此结构：{"name":"工具名称","params":{}}。

调用限制： 单次回复中禁最多出现3次 <tool> 调用。

自检机制： 如果不确定可用工具，请优先调用内置工具 listTools。

后续处理： 收到工具执行结果后，请继续按照此协议进行推理和回答。
